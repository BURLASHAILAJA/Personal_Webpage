# Shailaja Burla — Portfolio Website

A modern, single-page portfolio website showcasing expertise in **data engineering and analytics**, featuring a futuristic data-flow aesthetic with interactive animations.

## 🗂 Project Structure

| File         | Purpose                        |
|--------------|--------------------------------|
| `index.html` | Page structure & content       |
| `index.css`  | Styling & design system        |
| `script.js`  | Animations & interactivity     |

## 🧩 Sections

- **Navigation** — Sticky navbar with smooth scrolling & mobile hamburger menu
- **Hero** — Headline with stats (experience, projects, Fortune 500 clients) and an "Available for Projects" badge
- **About Me** — Bio with highlights (remote-ready, timezone-flexible, available immediately)
- **Core Expertise** — Cards for specializations (Data Modeling, Azure/ADF/Databricks, etc.)
- **Projects** — Portfolio showcase
- **Services** — Offered services
- **Contact** — Get-in-touch section

## ✨ Interactive Features

- Animated background canvas with connected data-flow particles
- Scroll-based reveal animations using Intersection Observer
- Proficiency bar animations for skill levels
- Typing effect for dynamic text
- Smooth scroll navigation

## 🎨 Design

- Dark theme with gradient accents
- Google Fonts (**Inter**)
- Glassmorphism-style floating cards
- Fully responsive design

## 🚀 Getting Started

1. Clone or download the project.
2. Open `index.html` in a browser, or start a local server:
   ```bash
   npx serve .
   ```
3. Visit `http://localhost:3000` in your browser.
